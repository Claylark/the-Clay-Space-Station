# 星壤空间站 | the Clay Space Station
## 项目介绍
这是一个基于 HTML + Tailwind CSS 构建的 个人主页，用于展示个人形象、兴趣爱好和社交账号，核心目的是「扩列」（寻找同好）。页面以《崩坏：星穹铁道》为核心兴趣点，融合了轨道交通、编程、地理研究等个人爱好，适配移动端和桌面端。
本项目基于《星壤使用许可协议》发布，使用需遵守该协议条款。

## 网页特点
1. 设计风格
梦幻渐变配色：以紫色（primary）和蓝色（rail）为主色调，搭配柔和渐变背景
交互效果：卡片悬浮上浮、导航高亮、平滑滚动、图片 hover 缩放等动态效果
响应式设计：完美适配手机、平板、电脑等不同设备
细节处理：自定义滚动条、阴影层次感、标签 hover 动画等提升质感
2. 核心功能模块
个人资料：展示 CN、基本信息、性格属性（ENFP）等
兴趣爱好：主推星穹铁道角色、核心爱好标签、关注领域、雷点说明
扩列专区：明确想认识的同好类型和可交流话题
联系方式：整合 QQ、小红书、B 站、LOFTER、GitHub 等社交平台
好友推荐：展示靠谱同好的主页链接，方便互相引流
属性历史：记录页面更新日志

## 页面结构
```plaintext
├── 导航栏（固定顶部，滚动自适应）
├── 英雄区域（头像、昵称、简介、行动按钮）
├── 主体内容
│   ├── 个人资料（基本信息 + 性格属性 + 备注）
│   ├── 兴趣爱好（主推角色轮播 + 爱好标签 + 关注领域 + 雷点）
│   ├── 想认识 & 交流话题（同好类型 + 可聊内容）
│   ├── 联系方式（多平台链接整合）
│   ├── 好友推荐（同好主页卡片）
│   └── 属性历史（页面更新记录）
└── 页脚（版权信息 + 个性签名）
```

## 技术栈
基础框架：HTML5 + CSS3
样式工具：Tailwind CSS v3（快速构建响应式界面）
图标库：Font Awesome 4.7（社交图标、功能图标）
字体：Inter（无衬线字体，清晰易读）
交互：原生 JavaScript（滚动监听、菜单切换、动画触发）

## 部署方法
1. 本地运行
下载项目代码压缩包并解压
直接打开 index.html 文件即可在浏览器中预览
2. 线上部署
方案 1：GitHub Pages（推荐）
将代码上传到个人 GitHub 仓库（需遵守许可协议）
仓库设置 → Pages → 选择分支（main）和目录（/root）
等待自动部署，访问仓库对应的 GitHub Pages 地址即可
方案 2：其他平台（Netlify、Vercel、阿里云等）
上传代码到对应平台
按照平台指引完成部署，无需额外配置（静态网页直接部署）
自定义指南
1. 修改个人信息
打开 index.html，搜索对应模块（如「个人资料」「联系方式」）
直接替换文本内容、链接、图片地址等
2. 更换图片
头像、角色图片等建议上传到安全的图床或代码仓库，使用稳定链接
替换 
```html
<img src="xxx" alt="xxx">
```
 中的 src 属性值

3. 调整样式
颜色修改：在 tailwind.config.js 中修改 colors 配置（primary、secondary 等）
字体调整：更换 fontFamily 中的字体名称（需先引入对应字体）
动画效果：修改 @layer utilities 中的过渡时间、缩放比例等参数
4. 添加新模块
在 <main> 标签内新增 <section> 区块
参考现有模块的结构，使用 Tailwind CSS 编写样式
在导航栏添加对应链接（桌面端 + 移动端）
